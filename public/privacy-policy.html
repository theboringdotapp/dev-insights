Privacy Policy

Last Updated: May 15, 2025

Welcome to DevInsight ("we," "us," or "our"). This Privacy Policy explains how we handle your information when you use our web application (the "Service"). By using the Service, you agree to the terms outlined in this policy.

Information We Access

When you authenticate with your GitHub account, we request permission to access certain information from your GitHub account solely for the purpose of providing the Service:

  • Pull Request Data: We access information about your pull requests, including titles, descriptions, commit messages, and related metadata.
  • Repository Information: We access basic information about repositories you contribute to for contextual analysis.
  • User Profile: We access your GitHub username and profile information to identify your contributions.

How We Use Information

  • Pull Request Analysis: We analyze your pull requests to generate insights about code quality, patterns, and development habits.
  • Activity Charts: We create visualizations of your GitHub activity based on pull requests and commits.
  • AI-Powered Feedback: When you choose to analyze specific PRs, we use AI to generate constructive feedback about your code quality and patterns.
  • Meta-Analysis: We identify recurring patterns across your pull requests to provide broader insights into your coding style and habits.

Data Storage and Security

We are committed to protecting your data. Here's how we handle storage and security:

  • Browser-Only Processing: DevInsight is a browser-only application. Your GitHub data and code are processed directly in your browser and are never stored on our servers.
  • Authentication Tokens: Your GitHub personal access token is stored securely in your browser's local storage and is never transmitted to our servers.
    • The token is used directly from your browser to communicate with GitHub's API.
    • All communication between your browser and GitHub's servers uses HTTPS encryption to protect data in transit.
  • AI Analysis: When you choose to analyze your code with AI:
    • Your code is sent directly from your browser to the AI provider (e.g., OpenAI, Anthropic) using your own API keys.
    • We do not store, see, or process your code or API keys on our servers.
    • The AI provider's privacy policy applies to how they handle your data during analysis.
  • Local Caching: Analysis results may be cached in your browser's local storage for performance reasons. This data remains on your device and is never transmitted to our servers.

Data Retention and Deletion

We believe in minimizing data retention. Here's our policy:

  • GitHub Data: Since we do not store your GitHub data on our servers, there is no server-side data to delete. Your data is fetched directly from GitHub's API to your browser during each session.
  • Browser Data: Data stored in your browser's local storage (such as authentication tokens, cached analysis results) can be cleared by:
    • Logging out of the application
    • Clearing your browser cache
    • Revoking the GitHub token from your GitHub account settings
  • Account Deletion / Revocation: You have full control over the application's access to your GitHub Account. You can revoke DevInsight's permissions at any time by deleting your personal access token in your GitHub security settings.

Third-Party Services

  • GitHub API: We access GitHub's API using your personal access token. Your interaction with GitHub is governed by GitHub's Privacy Policy.
  • AI Providers: If you choose to use AI analysis features, you will use your own API keys to communicate directly with providers like OpenAI, Anthropic, or Google. Their respective privacy policies apply to how they handle your data.
  • Analytics: We use Umami for simple, privacy-focused analytics (page views, feature usage) to improve the service. Umami does not use cookies and anonymizes IP addresses.

Your Choices and Rights

  • Token Management: You can delete your GitHub personal access token at any time through your GitHub security settings.
  • Local Data: You can clear locally stored data by logging out or clearing your browser cache.
  • AI API Keys: You have full control over your AI provider API keys and can choose whether to use the AI analysis features.

Open Source

DevInsight is open source. You can review the code on GitHub to verify how your data is handled.

Changes to This Policy

We may update this Privacy Policy. We will notify you of any changes by posting the new policy on this page. You are advised to review this Privacy Policy periodically for any changes.

Contact Us

If you have any questions about this Privacy Policy, please contact us via email at contact@theboring.app or by opening an issue on our GitHub repository.
